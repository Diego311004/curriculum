# Videos del Portafolio

Esta carpeta debe contener:

1. **presentation.mp4** - Video corto de presentación profesional (recomendado: máximo 2-3 minutos, formato MP4)
2. **presentation.webm** (opcional) - Versión WebM del mismo video para mejor compatibilidad

**Sugerencias para el video:**
- Presentación personal breve
- Demostración de proyectos
- Explicación de habilidades
- Duración: 1-3 minutos
- Resolución: 720p o 1080p
- Formato: MP4 (H.264) para máxima compatibilidad

**Herramientas para crear el video:**
- OBS Studio (grabación de pantalla)
- Canva (edición simple)
- DaVinci Resolve (edición profesional gratuita)
- Windows Video Editor (incluido en Windows 10/11)
