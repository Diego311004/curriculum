# Imágenes del Portafolio

Esta carpeta debe contener las siguientes imágenes:

1. **profile.jpg** - Tu foto de perfil profesional (recomendado: 400x400px o similar)
2. **about.jpg** - Imagen relacionada contigo o tu trabajo (recomendado: 800x600px)
3. **tech.jpg** - Imagen relacionada con tecnología o desarrollo (recomendado: 1200x600px)
4. **project1.jpg** - Imagen de proyecto de desarrollo web (recomendado: 600x400px)
5. **project2.jpg** - Imagen de proyecto de videojuegos (recomendado: 600x400px)
6. **project3.jpg** - Imagen de proyecto de IoT/Robótica (recomendado: 600x400px)

**Nota:** Puedes usar imágenes de stock gratuitas de sitios como:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)

O usar tus propias imágenes de proyectos reales.
